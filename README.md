# Student Mental Health Analysis

## Project Objective
The objective of this project is to clean and analyze a student mental health dataset using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Data Cleaning
- Checked missing values
- Filled missing values in Age column
- Removed duplicate records
- Saved cleaned dataset

## Visualizations
1. Gender Distribution
2. Depression Analysis
3. Anxiety Analysis
4. Age Distribution
5. CGPA Distribution

## Conclusion
- Dataset was cleaned successfully.
- Gender distribution was analyzed.
- Depression and anxiety levels were visualized.
- Age distribution of students was studied.
- CGPA distribution was analyzed.
